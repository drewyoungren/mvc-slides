Save scene in URL

A set of 3Demos objects (and some display settings) can be encoded into the URL to be bookmarked or shared as a simply web link.

  1. Go to 3demos.ctl.columbia.edu
  2. Click Objects → Add Object +
    and select vector
  3. Adjust components/location. Repeat to add more.
    • Optional: You can have any of the numeric fields depend on a continuous parameter t or a discrete parameter n. Try it.
  4. When the scene is ready, click the bar code icon ("Encode URL") at the bottom. (The page will reload.)
  5. Copy and paste the address in the location bar of your browser to share.
